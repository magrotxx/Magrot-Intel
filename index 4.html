Install on iPhone (iOS)
  1. Open this page in Safari (must be Safari, not Chrome)
  2. Tap the Share button (box with arrow) at the bottom of Safari
  3. Scroll down and tap "Add to Home Screen"
  4. Tap Add — Magrot Intel is now on your home screen like a native app
Install on Android
  1. Open in Chrome
  2. Tap the 3-dot menu → "Install app" or "Add to Home screen"
  3. Tap Install — done
Deploy to GitHub Pages (Always Online)
  1. Create a free account at github.com
  2. Create a new repository named magrot-intel
  3. Upload all files from this folder (index.html, manifest.json, sw.js, icons/*)
  4. Go to Settings → Pages → Source: main branch → Save
  5. Your app lives at yourusername.github.io/magrot-intel — always online, free forever
